npm

  • npm is Node Package Manager. It's a package manager for external modules (source for execute codes previously written)
  • In Hyper type npm init and it will bring the npm information
  • It will ask you the information of the file you are creating and suggest some entries. You click enter for to confirm
  • A new file will be created in Atom called package.json containing the information you just entered
  • Any other NPM package installed will also be shown in this file

Installing Packages from npm

  • Look for the command you want to run
  • The example is Superhero names, and the package is called superheroes
  • Check the instructions on how to install and use the packaged on npm
  • on Hyper, make sure you are in the correct directory and type the name of the package, exactly how it is spelled in npm
  • It will add a line of dependencies in the package.json file
  • In Javascript you can require the package using Var or Const to run the code
    1. Using var
        2. var superheroes = require('superheroes');
        var mySHName = superheroes.random();
        console.log(mySHName);
    2. Using const
        3. const superheroes = require('superheroes');
        superheroes.all;
        var i = superheroes.random();
        console.log(i)

    Express

    • It was build to avoid repetitive codes
    • To download it you go to Express Js and check how to install in Hyper
    • Currently, it is npm install express
    • After it is done, it will be listed in the package json file
    • When requiring express, every app.get is a ROUTE and every route can be a new page of the website
    • To require express you need to enter
      1. const express = require("express");
      2. const app = express();
      3. app.get("/", function(req, res){
      4. res.send("< h1>Text< /h1>")
      5. });
      6. app.get("/contact", function(req, res){
      7. res.send("< h1>Text< /h1>")
      8. });
      9. app.listen(3000, function(){
      10. console.log("Server started on Port 3000")
      11. });

    This instructions will help to require express

    The consts are refering to express

    App.get is refering to the local file /

    The app.get is how the Route is defined

    The function(req, res) determines what is the response once the serves requests access

    res.send is the response to the request done, and get rendered in the browser

    app.listen is referring to Port 3000

    The function in the app.listen is the callback
