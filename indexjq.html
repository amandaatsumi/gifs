    • Manipulating Style

    • You can create a style in the CSS for a class and add is to the JS with a class method as $("h1").addClass("class-name");
    • You can remove a CSS style with a class method as $("h1").removeClass("class-name");
    • To add more than one class you add in the same place $("h1").addClass("class-name other-class");
    • To check if the element had that class $("h1").hasClass("class-name"); and it will return True or False

    Add eventListeners

    • Click Function
      • $("h1").click(function(){ $("h1").css("color", "purple");});
    • To add it for a group of items (it runs through all buttons in the html):
          • $("button").click(function() {
          $("h1").css("color", "green"); });
    • To replace a element for the letter typed in the keyboard
        • $(document).keydown(function(event){
        $("h1").text(event.key); } );
    • To change a color of a element by mouse
        • $("h1").on("mouseover", function(){
        $("h1").css("color", "purple"); } );
    • Change the text with a click
        • $("h1").click(function(){
        $("h1").text("Te peguei de novo!");});
    • To add elements before a element
        • $("h1").before("< button>New")
    • After will add the button after H1
    • Prepend() will add the button at the beginning, inside the element
    • Append() will add the new button to the end, inside the element
    • To remove elements
        • $("button").remove;
    • To Hide an element: The element gets removed from the flow
        • $("h1").on("click", function() {
        ("h1").hide(); } ) ;
    • To Show an element: The element is added to the flow
        • $("h1").on("click", function() {
        ("h1").show(); } ) ;
    • To apply Toggle to an element
        • $("h1").on("click", function() {
        ("h1").toggle(); } ) ;
    • Other options are fadeOut, fadeIn, fadeToggle, slideUp, slideDown, slideToggle

        • $("h1").on("click", function() {
