CSS Positioning

  • The content determines initially the sizes of boxes
  • HTML determines the order the elements will be displayed
  • Children sit on top or parents: Headers comes before DIVs
  • Changing the position
    1. Static: HTML Default settings
    2. Relative: it allows images to move from the default and there are 4:
      1. Top > Bottom > Left > Right
      2. It replaces the elements that were there before, and it moves opposite to the name
      3. It is used in CSS inside a class: position: relative; left:20px
      4. It doesn't change the other elements positions
      5. The element moves in relation to the default position
    3. Absolute: it moves the images but it removes it from the original position
      1. It should impact the position of the other elements
    4. Fixed: It fix an object to the body even if you move your page down

CSS Display Property

  • Block: take the whole width of the page. Can change the width
    • P - Paragraph
    • Hs - Headers
    • DIV - Divisions
    • OL, UL, LI - List and list items
    • Forms
    • The Width can be changed
  • Inline makes all elements to be displayed side by side in different blocks
    • Span: (<'span') it is used to divide a block element in more blocks. It needs a class or ID. Can't Change the Width
    • Images ('img)')
    • Anchors (<'a')< /li>
  • Inline-Block: allows element to the same line and change the width. Like Images
  • None: hides the element

General Tags information

  • Inline tag: makes all elements to be displayed side by side
  • Span Tag: makes the selected part become a block
  • Display None: Hides the element and the block
  • Visibility hidden: hides the element but keeps its block position
  • Text-align: centre; needs to be entered in the CSS - it centralizes
  • CSS: font-size: 90px
    • 100% (add to the parent font size)= 16 px (not affected by parent font size) = 1em (add to the parent font size)
    • Any configuration in Headers add to the body font size except for px and rem
    • If the H1 changes, then the centralizing will not work
    • It needs to use Margin:
      • Top, Right, bottom, left
      • For Horizontal alignment: 0 auto 0 auto
