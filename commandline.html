Command Line

ls-List

  • It lists all items inside a directory
  • It gives the same view as when you look at a folder
  • If you want to open a lower or different level you type cd and the name of the folder you want
  • CD stands for change directory
  • If you start to type the folder name and click tab it will complete the type for you
  • If there are two folders' name that starts with the same letters, it will show you the options, and you add more letters to differenciate
  • If type ~ it will bring you to the original folder
  • You can also type all the way as shortcut:
      • cd Desktop/Second/CSS
  • If you use the UP key you can select all information previously typed
  • if you type .. (two dots) you will move back one level in the folders
  • If you want to edit any of the information in Hyper click Option/Alt and move to the position you want to edit
  • ALso, Command + A move the cursor to the beginnig and Command + E moves it to the end
  • Lastly, Command + U clears the entire line
  • pwd is printing work direct commend and shows the entire location of the folder that you are currently
  • The starter folder is the root

Creating Folders

  • Type mkdir FolderName that stands for Make Directory
  • To Create a File, inside the new folder (or any folder) type touch file-name.file-type
      • touch amanda.txt
  • To open files inside a folder, select the folder then type open file-name.file-type
  • To open a project on atom, when inside the directory you want to open type atom . (don't forget the space)
  • To open the file in a specific application/program you use -a app-name file-name.file-type
      • -a atom test.txt

Destructive Commands

  • To remove files type rm -i file-name.file type and confirm with Y
      • rm -i text.txt
  • If you want to delete all the files type rm *
  • To delete a folder you type rm -r folder-name/
